Basics of Web Scraping

If you've ever copied and pasted information from a webpage, you've done the same thing as a web scraper, but on a much smaller, manual scale. Unlike the slow process of manually collecting data, web scraping employs clever automation to collect information.

Web scraping consists of two components: a web crawler and a web scraper. The web crawler is the horse, and the scraper is the chariot. As if by hand, the crawler guides the scraper through the internet, finding and collecting the desired data.

The crawler

A web crawler is a piece of automation that scans and searches the internet for data by following links and analyzing them, similar to a human with too much spare time.

The scraper

A web scraper is a specialized tool used to rapidly and efficiently collect information from web pages. Web scrapers come in various designs and levels of complexity, depending on the task at hand. A key component of every scraper is its data locators, used to identify the data you want to extract from the HTML file.
